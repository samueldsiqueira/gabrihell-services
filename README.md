# Gabrihell Services

Landing page para serviços de hunt no Tibia.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Deploy na Vercel

1. Faça push do código para o GitHub
2. Importe o projeto na Vercel
3. A Vercel vai detectar Next.js automaticamente e fazer o deploy

## Configuração

Edite os links de contato em `components/Contact.tsx`:
- WhatsApp: substitua `5500000000000` pelo número real
- Discord: substitua `seu-servidor` pelo link do servidor
